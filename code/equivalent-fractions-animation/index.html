$$\frac{1}{2} = \frac{2}{4} = \frac{3}{6}$$
$$\frac{1}{2} = \frac{1 \times 2}{2 \times 2} = \frac{2}{4}$$
$$\frac{1}{2} = \frac{1 \times 3}{2 \times 3} = \frac{3}{6}$$

×2 both → same value. ×3 both → same value.
